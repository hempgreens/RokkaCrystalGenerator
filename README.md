# RokkaCrystalGenerator
雪の結晶の幅広六花のような意匠のアイコンを作成することができます。  
以下のリンクからアクセスしてください。

https://hempgreens.github.io/RokkaCrystalGenerator/

# 各種機能

## 色を変更
4つのカラーピッカーはそれぞれ以下の部分の色に対応しています。
- 背景
- 外側のふち
- 結晶の色
- 内側のふち

## 大きさを変更
2つのスライダーはそれぞれ以下の大きさに対応しています。
- 結晶の大きさ
- ふちの太さ

## ボタン
3つのボタンはそれぞれ以下の機能を持っています。

### デフォルトに戻す
デフォルトの色、大きさに戻します。

### ダウンロード
現在の幅広六花をpngで保存します。

### この幅広六花を共有
現在の幅広六花を再現するURLをクリップボードにコピーします。

常識や良識をもってご使用ください。
